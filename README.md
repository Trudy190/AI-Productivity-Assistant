# AI Workplace Productivity Assistant

## Project Overview
AI Workplace Productivity Assistant is a modern web application that helps professionals automate everyday workplace tasks using Artificial Intelligence. The application provides tools for email generation, meeting note summarisation, task planning, research assistance, and an AI chatbot.

## Features Implemented
- Smart Email Generator
- Meeting Notes Summarizer
- AI Task Planner
- AI Research Assistant
- AI Chatbot Interface
- Responsive Dashboard
- Sidebar Navigation
- Responsible AI Disclaimer

## Technologies Used
- Lovable AI
- React
- TypeScript
- Tailwind CSS
- GitHub

## Setup Instructions
1. Clone this repository.
2. Install project dependencies.
3. Run the development server.
4. Open the application in your browser.
